# Site Logo Folder

Place your website logo image file in this folder (e.g., `logo.png` or `logo.svg`).

Once you place the logo here, let me know its exact filename (e.g., `logo.png` or `logo.svg`) and I will link it in your website's header, footer, navigation bar, and email templates!

### Recommended Specifications:
- **Format:** `.png` (transparent background) or `.svg` (vector, highly recommended for crisp scaling)
- **Suggested filename:** `logo.png` or `logo.svg`
